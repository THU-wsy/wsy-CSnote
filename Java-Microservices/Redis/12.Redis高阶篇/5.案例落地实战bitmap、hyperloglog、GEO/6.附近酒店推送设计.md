# 附近酒店的推送

### 需求分析

推送当前位置，指定范围内的酒店信息

### 架构设计

Redis的新类型GEO

![](images/15.GEO介绍.jpg)

http://www.redis.cn/commands/geoadd.html

### 编码实现

关键点：GEORADIUS，以给定的经纬度为中心，找出某一半径内的元素
